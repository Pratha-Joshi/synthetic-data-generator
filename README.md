# Synthetic Data Generator

An AI-powered application that generates realistic synthetic tabular datasets based on user-defined requirements. The project uses **Meta Llama 3.2-3B Instruct** to create CSV datasets for various domains through a simple **Gradio** interface.

## Features

- Generate synthetic datasets for any domain
- Specify the number of rows and column names
- Outputs data in CSV format
- Simple Gradio-based interface

## Tech Stack

- Python
- Gradio
- Hugging Face Transformers
- Meta Llama 3.2-3B Instruct
- PyTorch
- BitsAndBytes

## Workflow

```
User Input → Llama 3.2 → Synthetic CSV Dataset
```
